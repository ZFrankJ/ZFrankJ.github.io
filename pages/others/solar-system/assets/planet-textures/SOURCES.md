# Planet Texture Sources

All texture files in this directory were downloaded from Solar System Scope's
free Solar Textures pack:

https://www.solarsystemscope.com/textures/

Solar System Scope describes the pack as real planetary maps in equirectangular
projection, based on NASA elevation and imagery data, with colors tuned against
spacecraft and Hubble imagery. The page distributes the pack under the Creative
Commons Attribution 4.0 International license.
